# dbcache

Generic database caching layer for Go with SQL template support and automatic cache population.

## Features

- **SQL Templating**: Parameterized query support with $1, $2 placeholders
- **Automatic Caching**: Transparent cache population on cache misses
- **Type Safety**: Generics support for any data type
- **Cache Control**: Configurable expiration and cleanup intervals
- **Custom Loaders**: Optional custom loader functions for complex data loading

## Installation

```sh
go get github.com/kaichao/gopkg/dbcache
```

## Quick Start

For comprehensive examples, see the [examples directory](examples/).

```go
import (
    "database/sql"
    "time"
    "github.com/kaichao/gopkg/dbcache"
)

func main() {
    db, err := sql.Open("postgres", "postgres://user:password@localhost/dbname")
    if err != nil {
        panic(err)
    }
    defer db.Close()

    // Initialize cache for user emails
    emailCache := dbcache.New[string](
        db,
        "SELECT email FROM users WHERE id = $1",
        5*time.Minute,  // Cache expiration
        10*time.Minute, // Cleanup interval
        nil,            // Use default SQL loader
    )

    // Get user email - first call queries database, second uses cache
    email, err := emailCache.Get(123)
    if err != nil {
        panic(err)
    }
    fmt.Println("User email:", email)
}
```

## Dependencies

- [github.com/patrickmn/go-cache](https://github.com/patrickmn/go-cache) - In-memory cache implementation
- Go 1.18 or higher (generic type support required)

## Error Handling

All errors are returned as-is from database operations or custom loader functions. No special error wrapping is applied, allowing callers to handle errors directly.

## Performance Considerations

- Cache keys are generated by formatting parameters with `fmt.Sprintf("%v", params)`
- Consider using more efficient key generation for high-throughput applications
- Default expiration and cleanup intervals should be tuned based on data freshness requirements
- Memory usage scales with cached data and expiration settings
